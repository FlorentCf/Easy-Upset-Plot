# Privacy Policy

Easy UpSet Plot does not transmit report data to external services.

## Data Handling

- The visual runs inside Power BI and renders the data provided by the host.
- The visual does not call external APIs.
- The visual does not use web requests to send report content outside Power BI.
- The visual does not persist report data outside the Power BI host environment.
