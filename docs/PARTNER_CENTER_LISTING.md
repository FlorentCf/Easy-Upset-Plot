## Long Description

Easy UpSet Plot is a Power BI custom visual for exploring set intersections and criteria combinations in a compact, high-performance layout.

It is optimized for pre-aggregated business data and is designed to feel responsive in both Power BI Desktop and Power BI Service. The visual combines top intersection bars, left set-size bars, and a dot matrix to help users understand overlap patterns across multiple 0/1 conditions without relying on heavy DOM rendering.

Easy UpSet Plot supports two analysis modes:

- Exact (true / false): classical UpSet semantics
- Inclusive (require / ignore): active conditions are required, inactive ones are ignored

This makes the visual useful both for strict set intersection analysis and for business-style criteria exploration.

The visual includes native Power BI selection behavior, native highlight rendering, keyboard focus support, context menu support, tooltips, and a modern formatting pane. It is built to stay readable, stable, and lightweight even in constrained report environments.
